# 🌟 Mindwell – Mental Wellness Platform

Mindwell is a modern full-stack platform to track moods, explore wellness content, and visualize mental health trends. Built with React, Node.js, and MongoDB.

## 🌈 Features

- 🔐 Secure Authentication – Login/Signup with JWT.
- 📊 Mood Tracker – Log moods & visualize patterns.
- 🎵 Wellness Content Feed – Integrated with YTMusic API.
- 📈 Interactive Charts – Dynamic visualizations using Recharts.
- ⏰ Scheduled Reminders – Node-cron background jobs.
- 🌐 Responsive UI/UX – Works on mobile and desktop.

## 🛠 Tech Stack

Frontend: React, TailwindCSS, React Router DOM, Recharts  
Backend: Node.js, Express, MongoDB, JWT, Node-cron, Axios  
Other: Nodemon, ESLint, Moment.js, Framer Motion

## ⚡ Installation

1. Clone the repo: 
git clone https://github.com/ishika506/Mindwell.git
cd Mindwell

2. Install dependencies: 
# Backend
cd backend
npm install

# Frontend
cd ../frontend
npm install

3. Create `.env` files for secrets (not pushed to GitHub)
